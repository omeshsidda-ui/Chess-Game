Deployment

  1. Push this file to your repo root (or to /docs for project site).
  2. Enable GitHub Pages in Settings → Pages → Branch: main, Folder: / (root) or /docs.
  3. Visit the published URL shown in Settings (or https://<username>.github.io/<repo>/).
